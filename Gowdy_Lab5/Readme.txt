To compile this program. Simply type make into the terminal while you are in the directory that the program files are contained in. 

To run the program with a specific file, type "./main 'filename'" i.e. ./main data.txt

You can interact through the program through the menu. If any improper commands are input, you'll have to exit the program with control-z

The program should tell you when it completes any of the actions. For any questions about the program, please email pgowdy@ku.edu
